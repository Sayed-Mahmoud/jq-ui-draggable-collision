jQuery UI Draggable Collision

Fork of Sean Cusack's JQuery UI Draggable Collision-plugin. It adds collision and protrusion detection to jQuery UI's Draggables.

Although Sean's original plugin is already relatively old, is better than other plugins I could find. Unfortunately it didn't work on recent jQuery (UI) versions. So I decided to fork and improve it by:

  • Making it compatible with recent jQuery and jQuery UI versions (> 1.10)
  • Better directory-structure
  • Adding bower support

I'm planning to add installation instructions and some demo's here in the future.
